# CountingDown
My app helps its users keep track of their events and special dates by doing the math for them, and telling them how many days are left before the events. It allows them to add notes as well as different tasks to these events, being able to mark as completed the tasks. It's a simple app that I found inspiration for in my own life, as I have myself counted the days before a special date more than once.

It has 3 screens, (1) the event collection view, where users can see all their events, (2) the detail view, which shows the information of each event and allows the user to edit said information, (3) the alarms and notification screens, which show the different options for these.

For the external API integration I added the posibility to sync the user's events with their calendar app. They can choose which events to sync.

I used both collection views and table views in the app, collection for the main screen showing all the events. And table views were included in one way or other in every other screen.

For the complex component of this app I let the users edit the data of the events, as well as saving such data in their devices. I also added the functionality of recieving notifications and alarms based on the user's choice for each event they create.
